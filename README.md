
# Covid-19 Tracker

This is a frontend web application to visualize Covid-19 Data and Statistics in India.


## Documentation

[Documentation](https://linktodocumentation)

  
## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/saifu-a/covid-tracker.git
```

Go to the project directory

```bash
  cd covid-tracker
```

Start the server

```bash
  start chrome index.js
```
